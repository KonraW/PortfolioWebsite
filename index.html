Quiz

Project created in C# in the MVVM architecture. The coded database of questions and correct answers is imported into the application. Then the user can choose from the list which quiz he wants to solve. When he presses start, time starts to fly and the user has to select the correct answers. At the end, the result achieved by the user appears.
