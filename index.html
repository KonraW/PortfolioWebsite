Film Base


Project created in C# in the MVVM architecture. The database of films is imported into the application, and then the user can filter the displayed elements by genre, title, etc. Additionally, he can give his own rating and mark the viewing, which is saved directly to the database.

Quiz


Project created in C# in the MVVM architecture. The coded database of questions and correct answers is imported into the application. Then the user can choose from the list which quiz he or she wants to solve. When he presses start, time starts to fly and the user has to select the correct answers. At the end, the result achieved by the user appears.

Game Civilization 2077


A turn-based strategy game, created by a 3-person team. Created using GDScript in the Godot as a school project. The player builds special buildings on a hexagonal board and recruits robots. Enemy buildings are scattered around him and enemy units come out of them. Robots can fight each other and destroy buildings. When one of the capitals falls, the game ends.
